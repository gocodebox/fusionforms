== Description ==
This plugin allows themes and other plugins to quickly and easily connect to the Infusionsoft PHP SDK via WordPress.

== Installation ==
1. Upload `infusionpress.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Navigate to the InfusionPress settings page
4. Add your Infusionsoft Application Credentials
